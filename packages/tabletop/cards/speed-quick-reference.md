```
1. SELECT    Both pick secretly (approach + location/setup)
2. REVEAL    Show choices, calculate modifiers
3. BATTLE    Both roll 2d6 + mods, higher wins
4. RESULT    Winner rolls for outcome
```

## GAME SETUP

**Each team needs:**
- Way to secretly select (fingers, cards, or dial)
- 2d6 (different colors help)

**Track:**
- Outs (3 per half-inning)
- Runners on base
- Score by inning

**Play 3, 5, 7, or 9 innings!**
