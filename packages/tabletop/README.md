## How to Print

1. **Quick Reference Booklet**: Open `quickref-booklet.html` in a browser and print (letter size, fold into quarters)
2. **Index Cards**: Print `cards/quick-reference-index-cards.pdf` on cardstock
3. **Play Mat**: Print `play-mat.md` or render to PDF

## What You Need to Play

- 2 six-sided dice (2d6)
- Quick reference cards (one for each player)
- Scorecard (optional)
- Play mat (optional)

## Game Summary

1. **SELECT**: Both players secretly choose approach + location
2. **REVEAL**: Show selections, calculate modifiers
3. **BATTLE**: Both roll 2d6 + modifiers, higher wins
4. **RESULT**: Winner rolls on outcome table

See `rules/cheddar-bob-rules-v5.md` for complete rules.
