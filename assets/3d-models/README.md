# 3D Printable Models

Game accessories for tabletop play.

## File Formats

- **STL**: Standard mesh format, widely supported
- **3MF**: Modern format with color/material data, better for multi-color prints

## Planned Models

### Base Markers
- Runner tokens for 1st, 2nd, 3rd base
- Different colors or numbered for batting order

### Trackers
- Out counter (0-2)
- Inning counter (1-9+)
- Score tracker

### Dice Accessories
- Dice holder/tray
- Dice tower (optional)

### Game Organizer
- Card holder
- Component tray

## Printing Guidelines

- Most parts designed for standard FDM printers
- Recommended layer height: 0.2mm
- Recommended infill: 15-20%
- Support may be needed for some models
