# Strikezone Assets

Shared assets for all platforms.

## Folders

### images/
Visual assets for game UI:
- Icons
- Backgrounds
- Team logos
- Card artwork

### 3d-models/
3D printable game accessories:
- `stl/` - STL format files
- `3mf/` - 3MF format files (with color/material data)

Ideas for 3D printed accessories:
- Custom dice holders
- Base markers (1st, 2nd, 3rd, home)
- Out trackers
- Score tracker
- Pitch/approach selection wheels

### audio/
Sound effects and music:
- Bat crack sounds
- Crowd noise
- Umpire calls
- Background music

### fonts/
Custom fonts for game UI:
- Scoreboard-style fonts
- Baseball-themed display fonts
